Projet: Backtest de l'analyse en 3 Touches en trading (Prologue)

1 - Extraire les fichiers de l'archive backtest_3T
2 - Ouvrir le fichier extrait avec vs code ou dans un terminal
3 - Via le terminal ou vs code, accéder au dossier backend, et exécuter la commande "pip install -r requirements.txt"
4 - Lancer le backend en exécutant la commande "uvicorn main:app --reload --host 0.0.0.0 --port 8000"
5 - Via le terminal ou vs code, accéder au dossier backend-interface, et exécuter la commande "npm install"
6 - Lancer le backend en exécutant la commande "npm start"
